Stanford Center for Cognitive an Neurobiological Imaging (CNI)
Peripheral Data Recording (PDR)

A python library and associated scripts for capturing all the peripheral 
data available in the CNI MR suite. Starts and stops recording peripheral data,
such as the eye tracker, the subject monitor video stream, and the physio data
stream from the LabJack.
